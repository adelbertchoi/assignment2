To run in Terminal (MAC):
1. Set path to ‘Final’ Folder
2. Type javac MiniNet.java
3. Type java -cp .:sqlite-jdbc-3.21.0.jar MiniNet

To run in eclipse:
1. Import entire project folder to eclipse
2. Go to project properties 
-> java build path 
-> libraries
-> add external jars
-> select ‘sqlite-jdbc-3.21.0.jar’
-> apply and close
3. Run MiniNet.java
